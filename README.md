# Sudoku
Проект для курсовой работы

## Сделано
* Нарисована таблица 9x9
* В таблицу записываются числа

## Необходимо доделать
* Добавление пустых ячеек
* Сделать заполненные ячейки не кликабельными
* Реализация событий нажатия на кнопки
* Добавить проверку правильности ввода числа в пустую ячейку
* Добавить условие выйгрыша игрока

## В планах
* Реализовать дополнительные activity (меню, правила игры и т.д)
* Возможно добавление уровней сложности
